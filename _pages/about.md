I am a Developer Tooling specialist with a strong focus on Microsoft technologies and GitHub. Over the years, I've supported large enterprise organizations in redesigning and modernizing their developer tooling landscapes. I've consistently seen that the tools developers rely on are often fragmented, outdated, or misaligned with delivery goals. My mission is to build streamlined, scalable, and future-proof platforms that enable fast feedback and high-quality software delivery.

My expertise lies in transforming developer platforms with the focus on GitHub and Azure DevOps, from isolated toolchains into integrated, capability-as-a-service platforms that make it easy for teams to onboard developers, applications, and systems. 

I'm passionate about reducing friction in software delivery through better tooling, automation, and intuitive developer experiences. The platforms I build are designed to fully support DORA capabilities such as Continuous Integration, Continuous Testing, and Continuous Delivery, while reducing cognitive load for development teams. Having coached teams on these practices, I bring not only technical expertise but also a deep understanding of how to implement them effectively in day-to-day workflows. I also lead initiatives to embed security and quality into delivery pipelines, using tools like SonarQube, GitHub Advanced Security (Code Scanning, Secret Scanning, and Dependabot), and Software Composition Analysis tools such as Renovate.

I'm deeply involved in the GitHub ecosystem and regularly deliver hands-on trainings on GitHub, GitHub Copilot, and GitHub Advanced Security. I'm particularly fascinated by the adoption phase of GitHub Copilot in enterprise settings — not just the technical rollout, but how teams evolve with AI-assisted development in practice.

Beyond consulting, I developed the Gitleaks extension for Azure DevOps, which helps teams detect and manage secrets in git repositories.
